[llmfit](https://github.com/AlexsJones/llmfit) written by Alex Jones can help you figure out which LLM model best fits your gear setup. Versatile tools with many settings.

Written in Rut, can be installed with `cargo` or `brew`

```bash
brew tap AlexsJones/llmfit
brew install llmfit
```

and then simply running

```bash
llmfit
```

For a top 3 recommended list you can run

```bash
llmfit recommend -n 3
```
