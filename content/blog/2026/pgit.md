The tool written by Oliver Seifert [pgit](https://oseifert.ch/blog/building-pgit) can load git history to a Postgres database in order to perform further analysis. Pgit was build as a demo which uses a compression library [xpatch](https://github.com/ImGajeed76/xpatch) under the hood.
