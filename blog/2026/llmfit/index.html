AlexsJones/llmfit

22 Feb, 2026

llmfit written by Alex Jones can help you figure out which LLM model best fits your gear setup. Versatile tools with many settings.

Written in Rut, can be installed with cargo or brew

brew tap AlexsJones/llmfit
brew install llmfit

and then simply running

llmfit

For a top 3 recommended list you can run

llmfit recommend -n 3
